# Reglas Globales de Desarrollo
   - Siempre que necesite documentación técnica, ejemplos de código o configuración de librerías, utiliza la herramienta Context7 (vía MCP) de forma proactiva.
   - Si pregunto por versiones específicas (ej. Spring Boot 3, FastAPI, Java 21), usa Context7 para evitar alucinaciones con versiones antiguas.
